# Easy Recycling

Progetto di User Experience incentrato sul riuso e sulla riparazione degli oggetti, con ricerca utenti, needfinding, prototipazione e testing.

## Contenuti del progetto

- `Relazione Finale.pdf`: documento principale del progetto.
- `Interviste Integrali.pdf`: trascrizioni complete delle interviste.
- `Questionario Easy Recycling (Risposte).xlsx`: dataset delle risposte al questionario.
- `Testing Prototipi.pdf`: risultati dei test sui prototipi.
- `Prototipi/Low Fidelity/`: paper prototype v1 e v2.
- `Prototipi/High Fidelity/ReadMe.txt`: link ai prototipi high-fidelity su Figma.

## Link utili

- Questionario: disponibile in `ReadMe.txt`.
- Prototipi high-fidelity (Figma): disponibili in `ReadMe.txt` e `Prototipi/High Fidelity/ReadMe.txt`.
